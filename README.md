# Memory Game Assessment
A memory matching game designed to test your focus and pattern recognition made for an assessment.
---
## How to Play
1. **Flip one card** to reveal its symbol.
2. **Flip another card** and try to find its matching pair.
3. If the two cards **match**, they remain face-up.
4. If they **do not match**, they will flip back over after a short delay.
5. Continue until **all pairs are matched**.
